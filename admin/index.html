Archivo descargado — ahora publica los cambios
  1. Busca el archivo descargado en tu carpeta Descargas.
  2. Cópialo (reemplaza) en wayra/data/ dentro de tu repositorio local.
  3. Abre una terminal en esa carpeta y ejecuta:
    git add data/ && git commit -m "actualizar contenido" && git push
  4. Vercel detecta el push y redespliega automáticamente en ~1 minuto.

Bienvenido al panel

🚀 Flujo para publicar cambios en Vercel
1 Elige la sección (Experiencias, Planes, Hoteles o Configuración) en el menú lateral.
2 Haz clic en ☁️ Cargar datos del sitio para traer el contenido actual.
3 Edita el contenido y haz clic en 💾 Guardar — se descargará el archivo JSON actualizado.
4 Copia el archivo descargado a la carpeta data/ de tu repositorio y haz git push.
✓ Vercel redespliega automáticamente en ~1 minuto. ¡Listo!
